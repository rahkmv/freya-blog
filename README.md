# Freya Portfolio Blog

Mobile Legends Bang Bang o'yinidagi Freya qahramoni uchun portfolio blog sayti.

## Fayllar Tarkibi

1. `index.html` - Asosiy HTML sahifa (Bootstrap 5 bilan)
2. `style.css` - Barcha CSS stillar
3. `script.js` - Barcha JavaScript kodlar
4. `data.json` - Ma'lumotlar fayli
5. `assets/` - Rasmlar papkasi

## O'rnatish

1. Barcha fayllarni bir papkaga yuklang
2. `assets` papkasiga Freya rasmlarini joylang
3. Brauzerda `index.html` faylini oching

## Xususiyatlar

- Bootstrap 5 responsiv dizayn
- Smooth scroll navigation
- Dynamic content loading
- Mobile friendly
- Dark theme with golden accents
- Hero stats and skills display
- Blog posts section

## O'zgartirishlar

1. O'z NIKingizni `index.html` faylida qidiring va o'zgartiring
2. `data.json` faylida statistikani o'zgartiring
3. `assets` papkasiga o'zingizning rasmlaringizni qo'shing
4. CSS ranglarini `style.css` faylidagi `:root` bo'limida o'zgartiring
